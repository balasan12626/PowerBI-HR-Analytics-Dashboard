This project features a comprehensive **Power BI Dashboard** designed to analyze employee attrition within an organization. By leveraging the **IBM HR Analytics Employee Attrition & Performance dataset**, we identify key factors contributing to employee turnover, allowing HR departments to take data-driven actions to improve retention and employee satisfaction.

The dashboard provides a high-level overview of attrition rates, demographic breakdowns, and satisfying scores, deep-diving into job roles, income levels, and work-life balance correlations.
